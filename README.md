# Sistema de Gestión de Tareas

Este proyecto es una aplicación sencilla para organizar tareas diarias.
Permite crear, editar y marcar tareas como completadas.
Fue desarrollado como práctica académica del curso.

## Instalación

1. Clonar el repositorio
2. Instalar dependencias con `npm install`
3. Ejecutar el proyecto con `npm start`

## Tecnologías utilizadas

- JavaScript
- Node.js
- Visual Studio Code

Más información en [este enlace](https://monday.com/blog/es/gestion-del-trabajo/control-de-tareas/).
